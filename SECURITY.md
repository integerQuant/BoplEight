# Security

BoplEight processes packets from Steam peers and its installer writes files into the selected game directory. Security-sensitive changes require particular care around sender authorization, packet length ownership, path traversal, reparse points, and elevated installer behavior.

Report suspected vulnerabilities privately to the repository owner before opening a public issue. Include the affected BoplEight version, reproduction steps, and expected impact. Do not include Steam credentials, private lobby data, or proprietary game files.
